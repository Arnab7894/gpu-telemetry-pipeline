{{- if .Values.apiGateway.enabled }}

API Gateway is available at:
  {{ .Release.Name }}-api-gateway:{{ .Values.apiGateway.service.port }}

To access the API Gateway from outside the cluster:

  1. Port-forward the service:
     $ kubectl port-forward service/{{ .Release.Name }}-api-gateway {{ .Values.apiGateway.service.port }}:{{ .Values.apiGateway.service.port }}

  2. Access the API:
     $ curl http://localhost:{{ .Values.apiGateway.service.port }}/health
     $ curl http://localhost:{{ .Values.apiGateway.service.port }}/api/v1/gpus

  3. View Swagger documentation:
     http://localhost:{{ .Values.apiGateway.service.port }}/swagger/index.html

{{- end }}
